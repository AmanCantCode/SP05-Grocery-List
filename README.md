# SP-05-Grocery-List

Shared Grocery List App
A Flutter + Supabase Senior Project

A cross-platform grocery list application built using Flutter and Supabase.
It allows families, roommates, or teams to collaborate on shared shopping lists in real time.
Users can create groups, invite members, manage grocery items, and view nutritional data through third-party integration — all within a clean and modern UI.

# Features
## Shared Grocery Lists

- Create or join groups with multiple members
- Add, edit, and delete items from the shared grocery list
- Items update instantly for all group members
- Checked-off items persist using Supabase database updates

## User Authentication & Accounts

- Secure email/password authentication via Supabase Auth
- User profile page with editable display name
- Supports multiple groups under one account

## Nutrition Lookup Integration

- Integrated third-party API allows users to:
- Search for food items
- View detailed nutritional information
- Add items to the grocery list based on search results

## Modern UI/UX

- Built with Flutter’s Material 3
- Clean custom UI for dialogs, input fields, and list tiles
- Adaptive layouts for mobile devices
- Optional dark-mode support
